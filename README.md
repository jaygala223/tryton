# Tryton: Learning Triton Lang

**Author:** Jay Gala

A collection of Triton kernels for learning GPU programming in Python.

## What's Triton?

Triton is a language for writing efficient CUDA kernels in Python without dealing with raw CUDA code.

## Contents

- `add.py` - Vector addition kernel (starter example)
- `benchmarking.py` - Performance measurement utilities

## Quick Start

```bash
# Install Triton
pip install triton

# Run vector addition example
python3 add.py
```
